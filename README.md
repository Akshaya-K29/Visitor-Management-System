# Visitor Management System

## Project Description
A full-stack Visitor Management System developed using Spring Boot, HTML, CSS, JavaScript, and MySQL. The system helps manage visitor registration, approvals, check-in, check-out, and dashboard records.

## Technologies Used
- Frontend: HTML, CSS, JavaScript, Bootstrap
- Backend: Spring Boot (Java)
- Database: MySQL
- Build Tool: Maven

## Features
- Visitor Registration
- Login Authentication
- Admin Dashboard
- Visitor Approval
- Check-In and Check-Out
- QR Pass Generation
- File Upload
- Visitor Records Management

## Project Structure
```
src/
pom.xml
mvnw
mvnw.cmd
```

## Database
Create a MySQL database named:

```
visitor_management
```

Update the database credentials in:

```
src/main/resources/application.properties
```

## Run the Project
1. Clone the repository.
2. Open the project in Spring Tool Suite (STS) or IntelliJ IDEA.
3. Configure MySQL.
4. Run `VisitorManagementApplication.java`.
5. Open the frontend in your browser.
